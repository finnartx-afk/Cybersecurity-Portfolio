# Python Port Scanner

A beginner cybersecurity project written in Python using sockets.

## Features

- Accept a hostname from the user
- Scan multiple TCP ports
- Detect open and closed ports
- Handle invalid hostnames
- Stop scanning when the host cannot be found

## Skills Practiced

- Python
- Socket programming
- TCP ports
- Networking fundamentals
- For loops
- If statements
- Try and except
- Error handling
- Debugging
